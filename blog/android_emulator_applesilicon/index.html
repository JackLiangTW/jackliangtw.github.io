Apple Silicon 上設定 Android 模擬器與 Magisk 注入

Apple Silicon(M1/M2/M3)上跑模擬器

Android Studio AVD

  • 用 Android Studio 內建 AVD Manager 建 arm64 image。
  • 新版 Android Studio 直接支援 Apple Silicon,不用特別處理。

android-emulator-m1-preview(舊)

  • 官方唯一支援 Apple Chip 的簡易模擬器(早期替代品),有 Chrome / WebView。
  • 限制:不能直接下載 APK(WebView 攔),改從雲端硬碟下載,或用「APK Editor Studio」(https://qwertycube.com/apk-editor-studio/)從 Mac 推進模擬器。

macOS 設定

  • sudo spctl --master-disable → 允許「任何來源」安裝 App(Gatekeeper off)。
  • adb 路徑:
export PATH=$PATH:$HOME/Library/Android/sdk/platform-tools

AVD Root 化(Magisk 注入)

API ≤ 27 才吃得下 rootAVD 的注入法。

  1. Android Studio 建 AVD,system image 挑 API 27、arm64-v8a。
  2. 啟動模擬器一次(讓 image 解壓)。
  3. 取 rootAVD:https://github.com/newbit1/rootAVD。
  4. 跑注入:
./rootAVD.sh system-images/android-27/default/arm64-v8a/ramdisk.img
  1. Cold boot AVD。裝 Xplore file manager,開啟 / 目錄時會跳出 Magisk 請求 root → 成功。

LSPosed 與 Shamiko

  • LSPosed:Magisk 模組系統的 Xposed 替代品;Magisk App → Modules → 找 LSPosed → Install → reboot。官方:https://magiskcn.com/lsposed-install。
  • Shamiko / HideRoot:隱藏 root 逃過 SafetyNet / Play Integrity 檢測。在 Magisk DenyList 設定要藏 root 的 app。某些金融 App(GCash 等)仍會被檢出,要搭配關閉 Shamiko whitelist mode 試看看。

APK 版本碼

  • versionCode:App 升級比對的整數值(新值必須 > 舊值)。
  • xapk = ZIP + 多個 split APK + manifest.json;manifest.json 的 version_code 是 最高的(≥ 任何子 apk 的值)。
  • 在 macOS 把 .xapk 改成 .zip 直接解壓即可取得原始 apk 檔案。

FC 客戶端開發必備

  • Android Studio + Kotlin 基礎。
  • 模擬器(雷電、Genymotion 或 AVD)。
  • Magisk + LSPosed(Hook 框架)。
  • YukiHookApi(Kotlin hook 框架)官方文件:https://highcapable.github.io/YukiHookAPI/en/guide/home.html。
  • 備選 Hook:Xposed 原生、Frida。
